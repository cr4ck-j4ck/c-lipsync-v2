# C-Lipsync V2

LAN clipboard sync and remote text helper for two computers on the same WiFi/LAN.

## Start

```bash
pnpm start
```

## Help

```bash
pnpm run help
pnpm start -- --help
```

Inside remote command mode:

```text
.help
```

See [docs/USAGE.md](docs/USAGE.md) for command examples, code-pasting workflow, troubleshooting, and why `.setclip` is usually the best option for formatted code.
